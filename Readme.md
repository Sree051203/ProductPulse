## âœ¨ Project Overview
**ProductPulse** is an end-to-end analytics project designed to analyze customer reviews, classify sentiment, and offer actionable insights into product performance and inventory health.

The pipeline:
- Extracts & cleans data  
- Processes reviews  
- Classifies sentiment using `VADER`  
- Generates category-level summaries  
- Provides a Streamlit dashboard for visualization  

This project is ideal for **students, ML beginners, and portfolio building**.
